### Description

This program aims to generate Dedukti compatible files from xml files of the [Termination Problem Data Base](http://cl2-informatik.uibk.ac.at/mercurial.cgi/TPDB)

### Compilation

In order to compile, simply do `make`.

You will need:
 - `OCaml >= 4.02`,
 - `OCamlBuild`,
 - `xml-light`.
